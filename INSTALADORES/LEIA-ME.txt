🚀 COMO INSTALAR:

1. Dê duplo clique em "NeuroGame Launcher Setup 1.0.1.exe"

2. O instalador irá:
   ✓ Permitir escolher o local de instalação
   ✓ Criar atalho na Área de Trabalho
   ✓ Criar atalho no Menu Iniciar
   ✓ Instalar todos os componentes necessários

3. Após a instalação, o launcher será aberto automaticamente

4. Faça login com suas credenciais NeuroGame

📋 REQUISITOS DO SISTEMA:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• Sistema Operacional: Windows 7 ou superior
• Memória RAM: Mínimo 2GB (recomendado 4GB)
• Espaço em Disco: 200MB para o launcher + espaço para jogos
• Internet: Conexão estável para download de jogos


🎮 APÓS A INSTALAÇÃO:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. O NeuroGame Launcher ficará disponível em:
   • Atalho na Área de Trabalho
   • Menu Iniciar → NeuroGame Launcher
   • C:\Users\[SeuUsuário]\AppData\Local\Programs\neurogame-launcher

2. Na primeira execução:
   • Faça login com seu usuário e senha
   • Navegue pela biblioteca de jogos
   • Clique em um jogo para baixar e instalar

3. Os jogos serão instalados em:
   • Pasta padrão: C:\NeuroGame\Jogos\
   • Você pode alterar nas configurações
